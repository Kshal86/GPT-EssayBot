
EssayBot Chatbot


By: Kushal Prajapati

Description:
The EssayBot Chatbot is a simple chatbot application that utilizes the OpenAI GPT-3 language model to generate essay-like responses based on user input. It provides an interactive interface for users to engage in conversations and receive detailed essay responses.

Purpose:
The purpose of this project is to demonstrate the capabilities of the OpenAI GPT-3 language model and showcase its potential applications in generating informative and detailed responses. The EssayBot Chatbot can be used for educational purposes, idea generation, or as a tool to assist in writing essays or articles.

Implementations:
- The chatbot user interface is built using the tkinter library in Python, providing a visually appealing and interactive experience.
- The OpenAI GPT-3 language model is utilized through the OpenAI Python library to generate essay-like responses.
- The chatbot logic handles user input, sends requests to the OpenAI API, and processes the model's responses.

Welcome to the EssayBot Chatbot! This is a simple chatbot application that uses the OpenAI GPT-3 language model to provide responses to user input. The chatbot can generate essay-like responses based on the user's queries.

Requirements:

- Python 3.x
- tkinter (typically comes pre-installed with Python)
- OpenAI Python library

Installation:

1. Clone the following repository: `git clone https://github.com/Kshal86/essaybot.git`
2. Install the required packages: pip install the following- Openai, Tkinter, PIL
3. Obtain an OpenAI API key from [OpenAI](https://openai.com/) and replace the placeholder API key in `chatbot_logic.py`.

Usage

To start the EssayBot Chatbot, run the following command:
python main.py
or use your preffered IDE to run the main.py file
This will launch the chatbot user interface, where you can interact with the chatbot.

Customization

- You can customize the appearance of the chatbot user interface by modifying the styles and images in the `chatbot_ui.py` file.
- The chatbot's behavior and responses can be modified by adjusting the prompts and parameters in the `chatbot_logic.py` file.
